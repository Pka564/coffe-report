# Отчет о потреблении кофе

Консольный скрипт для обработки CSV-файлов с данными о студентах и формирования отчета по медианным тратам на кофе.

## Функциональность

- Чтение одного или нескольких CSV-файлов
- Расчет медианы трат на кофе (`coffee_spent`) по каждому студенту
- Объединение данных из всех файлов
- Сортировка результатов по убыванию трат
- Вывод результата в консоль

## Формат входных данных

CSV-файлы должны содержать следующие колонки:

student,date,coffee_spent,sleep_hours,study_hours,mood,exam

Пример строки:

Алексей Смирнов,2024-06-01,450,4.5,12,норм,Математика

## Запуск

Из папки с проектом:

py main.py --files math.csv physics.csv --report median-coffee

или:

python main.py --files math.csv physics.csv --report median-coffee

## Пример вывода

Student | Median Coffee

Иван Кузнецов | 650
Алексей Смирнов | 500
Мария Соколова | 120

## Тестирование

Установить pytest:
py -m pip install pytest

Запустить тесты:
py -m pytest

## 📌 Особенности реализации

- Используется стандартная библиотека Python
- Логика расчета вынесена в отдельную функцию
- Поддерживается обработка нескольких файлов одновременно
- Код легко расширяется для добавления новых отчетов
